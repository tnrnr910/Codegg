## 1. 프로젝트 개요

- 프로젝트 제목
  - Codegg
- 프로젝트 소개
  - 프로젝트 기획 이유 : 이 프로젝트는 개발자들끼리의 의사소통 및 정보공유를 위해 기획했습니다.
  - 프로젝트 핵심 서비스 :  질문 및 답변 게시판을 통한 효율적인 정보 공유 및 프로젝트/스터디 게시판을 통한 소모임 인원 확보.
